# /cc-whatsapp:start — launch the router for this project

## Steps

1. **Verify prerequisites.** Read these files; abort with a clear message if missing:
   - `.claude/cc-whatsapp/config.json` (must have `account`)
   - `~/.wacli/config.yaml` (must list that account — confirm pairing happened)

2. **Generate per-project run.command.** Write it to `.claude/cc-whatsapp/run.command`:

   ```bash
   #!/bin/zsh
   # Per-project cc-whatsapp router launcher. Generated by /cc-whatsapp:start.
   PROJECT="<absolute-path-to-current-cwd>"
   REPO="<absolute-path-to-${CLAUDE_PLUGIN_ROOT}/..>"
   cd "$PROJECT" || exit 1

   # Kill any prior router instance for THIS project.
   pkill -9 -f "router.ts.*${PROJECT}" 2>/dev/null
   sleep 1

   # Self-cleanup: close stale Terminal windows running this project's router.
   TITLE="cc-whatsapp · <project-basename>"
   osascript >/dev/null 2>&1 <<APPLESCRIPT
   tell application "Terminal"
     repeat with w in windows
       try
         if (busy of selected tab of w is false) and ((custom title of selected tab of w) is "${TITLE}") then
           close w saving no
         end if
       end try
     end repeat
   end tell
   APPLESCRIPT
   printf '\033]1;%s\007' "${TITLE}"
   printf '\033]2;%s\007' "${TITLE}"

   export CC_WHATSAPP_PROJECT_DIR="$PROJECT/.claude/cc-whatsapp"
   export CC_WHATSAPP_BIN="$REPO/bin/cc-whatsapp"

   echo "════════════════════════════════════════════════════════════════════════"
   echo "  cc-whatsapp router · ${TITLE}"
   echo "  project:  $PROJECT"
   echo "  binary:   $CC_WHATSAPP_BIN"
   echo "  Ctrl-C to stop · close window to kill"
   echo "════════════════════════════════════════════════════════════════════════"

   exec bun "$REPO/plugin/router.ts"
   ```

   Resolve `<absolute-path-to-current-cwd>`, `<absolute-path-to-${CLAUDE_PLUGIN_ROOT}/..>`, and `<project-basename>` with real paths.

3. **chmod +x** the run.command.

4. **`open .claude/cc-whatsapp/run.command`** — macOS Terminal pops a new window with the live router console.

5. **Wait 3 seconds, then verify** the router process started:
   ```bash
   pgrep -fl "router.ts.*<project-path>"
   ```
   Show the PID to the user.

6. **Print:**
   ```
   ✓ Router started in a new Terminal window.
     Watch live: tail -f .claude/cc-whatsapp/trace.log
     Stop:       /cc-whatsapp:stop  (or close the Terminal window)
   ```

## Notes

- Per project = per Terminal window. Each project has its own router.
- The router stays alive even if you close Claude Code — that's a feature.
- If a stale router from a prior session is still running for this project, the `pkill -9` in run.command kills it before starting fresh.
